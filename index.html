Word Salad

This game will help teach you the difference between a noun, an adjective and a verb. We'll start with a few questions, and then we'll start building sentences.

Here is a reminder of the basics...

A noun

(n) is a...

An adjective

(a) is a...

A verb

(v) is a...

For example, can you see where the noun is, where the adjective is and where the verb is in the following sentence?

The black cat runs up the street.

(n) cat

(a) black

(v) runs
